📝 Automating Attendance with Face Recognition: A Final-Year Project Journey

👋 Introduction

Remember the old-school roll calls in classrooms? Tedious, time-consuming, and prone to errors. What if we could automate this process using technology that recognizes faces and marks attendance in real-time?

That’s exactly what we set out to do in our final-year engineering project: an automated attendance system powered by face recognition and machine learning. This blog post walks you through our journey—from concept to implementation—highlighting the tech stack, challenges, and results.

🧠 The Problem We Solved

  • Manual attendance systems are inefficient.
  • They waste valuable class time.
  • Are prone to manipulation.
  • Require human supervision.

Our goal was to build a contactless, real-time, and accurate attendance system using facial recognition.

🛠️ Tools & Technologies

🔧 Hardware

  • Raspberry Pi 3 B+
  • Raspberry Pi NoIR Camera V2
  • USB to TTL Converter

💻 Software

  • Python 3
  • OpenCV (for face detection and recognition)
  • LBPH (Local Binary Patterns Histogram)
  • Haar Cascade Classifier
  • Raspbian OS
  • VNC Viewer & FileZilla

🧪 How It Works

  1. Face Dataset Creation: Students enroll by capturing multiple images of their face.
  2. Training the Recognizer: Using OpenCV’s LBPH algorithm, we train a model on the dataset.
  3. Real-Time Recognition: The camera captures faces and marks attendance if a match is found.

🧬 System Architecture

  • Face Detection: Haar Cascade Classifier
  • Feature Extraction: LBPH
  • Recognition: Matching against trained dataset
  • Attendance Logging: CSV file generation
