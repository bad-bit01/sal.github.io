Automated Attendance System using Face Recognition and Machine Learning

       

Final Year Project Summary

       

Introduction

       

This project explores the development of an automated attendance system that leverages face recognition and machine learning to streamline and secure the process of recording student attendance.

       

Key Technologies Used

       
               
  • Python
    •            
  • OpenCV
    •            
  • Face Recognition Library (dlib)
    •            
  • Machine Learning Algorithms
    •            
  • SQLite Database
    •        
       

System Design

       

The system is designed to capture real-time video input, detect and recognize faces, and log attendance data into a database. It includes modules for face detection, face encoding, and attendance logging.

       

Implementation Steps

       
               
  1. Data collection and preprocessing of student face images.
    2.            
  2. Training the face recognition model using encodings.
    3.            
  3. Real-time face detection and recognition using webcam input.
    4.            
  4. Logging recognized faces with timestamps into the database.
    5.            
  5. Generating attendance reports.
    6.        
       

Results

       

The system demonstrated high accuracy in recognizing registered faces and successfully automated the attendance process. It reduced manual errors and improved efficiency in classroom environments.

       

Conclusion

       

This project showcases the practical application of computer vision and machine learning in educational settings. It provides a scalable and secure solution for automating attendance using facial biometrics.
